Cascade

CSS (Cascade style sheet) aplica las propiedades CSS en cascada, cualquier propiedad css definido anteriormente o despues sobre el mismo elemento. Se aplican realizando un efecto en cascada. Esta forma de aplicar estilos se le llama cascade. Y sigue los siguientes pasos:

  1. Posición de lo estilos
    • Proximidad de los elementos a los estilos definidos en el codigo
  2. Especificidad,
    • ¿Que elemento hace mejor match con el selector de una regla de CSS?
  3. Origin
    • Del menos al más especifico --> user agent, local user (SO o extension), authored, authored with !important, local user (SO o extension) with !important, usear agent with !important
  4. Importance
    • Del menos al más importante --> normal, animation, !important, transition
